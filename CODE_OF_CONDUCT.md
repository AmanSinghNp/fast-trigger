# Code of Conduct

## Our commitment

We are committed to providing a friendly, safe, and welcoming environment for all contributors.

## Expected behavior

- Be respectful and constructive.
- Assume good intent and communicate clearly.
- Accept feedback gracefully.
- Focus on what is best for the project and community.

## Unacceptable behavior

- Harassment, insults, or discriminatory language.
- Personal attacks or trolling.
- Publishing private information without consent.
- Any conduct that is inappropriate in a professional setting.

## Reporting

To report unacceptable behavior, open a private security report or contact the maintainers through repository channels.

Project maintainers may remove, edit, or reject comments, commits, issues, and other contributions that violate this Code of Conduct.
