# Security Policy

## Reporting a vulnerability

Please report suspected vulnerabilities privately.

- Do not open public issues for security problems.
- Open a private GitHub security advisory at:
  `https://github.com/AmanSinghNp/fast-trigger/security/advisories/new`
- Include affected versions, impact, and reproduction steps.
- If possible, include a minimal proof of concept.

We will acknowledge reports as quickly as possible and work on a fix.

## Disclosure policy

- We aim to validate reports promptly.
- We will coordinate responsible disclosure with reporters.
- Security fixes may be released before full technical details are published.
